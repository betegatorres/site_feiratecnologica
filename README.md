# Assédio Moral - Site Educativo

Trabalho interdisciplinar sobre assédio moral desenvolvido pelos alunos do 1º ano do Ensino Médio Integrado ao Técnico em Informática para Internet.

## 📋 Sobre o Projeto

Este site foi criado com o objetivo de informar, educar e conscientizar sobre o assédio moral, mostrando suas consequências, formas de enfrentar e como expressar tudo aquilo que se passa. O projeto visa mostrar para quem comete assédio que está desestruturando não só uma pessoa, mas todos que estão à sua volta.

## 🎯 Objetivos

- Aumentar a consciência pública sobre o assédio moral
- Combater desinformações sobre o tema
- Mostrar que quem passa por isso nunca está sozinho
- Orientar sobre como buscar ajuda psicológica
- Ensinar a se afastar de situações que causam repressão ou ataque

## 🚀 Como Executar o Projeto

### Pré-requisitos

- Node.js instalado (versão 18 ou superior)
- npm, yarn ou pnpm instalado

### Instalação

1. Extraia o arquivo ZIP do projeto
2. Abra o terminal na pasta do projeto
3. Instale as dependências:

\`\`\`bash
npm install
# ou
yarn install
# ou
pnpm install
\`\`\`

### Executar em Desenvolvimento

\`\`\`bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
\`\`\`

Abra [http://localhost:3000](http://localhost:3000) no navegador para ver o site.

### Gerar Versão de Produção

\`\`\`bash
npm run build
npm run start
\`\`\`
